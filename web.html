Basic Usage

import ( inventory "github.com/saichler/l8inventory/go/inv/service" "github.com/saichler/l8types/go/ifs" ) // Activate inventory service serviceName := "inventory" serviceArea := byte(0) primaryKey := "id" elemType := &YourProtobufMessage{} vnic.Resources().Services().Activate( inventory.ServiceType, serviceName, serviceArea, resources, listener, primaryKey, elemType, )

Supported Operations

POST

Add new inventory items to the distributed cache with automatic validation and indexing.

PATCH

Update existing inventory items with delta changes and real-time notifications.

GET

Query and retrieve inventory data using SQL-like syntax with advanced filtering.

DELETE

Remove inventory items with proper cleanup and cascade operations support.
